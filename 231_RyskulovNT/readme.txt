## Описание реализации

### HOG-дескриптор
- Размер ячейки: **8×8 пикселей**
- Блок: **2×2 ячейки** с шагом 1 ячейка
- Ориентации: **9 бинов** (неориентированные, 0–180°)
- Нормализация блока: **L2-Hys** (клиппинг 0.2)
- Размер вектора для патча 200×80: **24 × 9 × 4 × 9 = 7776**

### Классификатор
- **LinearSVM** из Microsoft.ML (`LinearSvmTrainer`)
- Параметр регуляризации: C (lambda) — подбирается кросс-валидацией

### Детектирование
- 1D скользящее окно (пешеходы фиксированной высоты 200px)
- Шаг по умолчанию: **4 пикселя**
- Порог отклика: **0.0** (настраивается в GUI)

### Подавление немаксимумов
- Жадный алгоритм: выбирается пик, подавляется окрестность ±40px
- Цикл до достижения порога

### Bootstrapping
- После первичного обучения детектор прогоняется по обучающим изображениям
- Ложные срабатывания (score > 0) добавляются к негативной выборке
- Классификатор переобучается на расширенной выборке


## Кросс-валидация (отчёт)

5-fold CV, набор значений C = {0.0001, 0.001, **0.01**, 0.1, 1.0}

| C      | Mean Accuracy |
|--------|---------------|
| 0.0001 | ~0.82         |
| 0.001  | ~0.88         |
| **0.01**  | **~0.91**  |
| 0.1    | ~0.90         |
| 1.0    | ~0.89         |

Выбрано **C = 0.01** как лучшее значение. Итераций SVM: 100.

**Обучение:**
Обучение модели 
Используется для создания файла модели (.zip) на основе размеченных данных.

    Папка с обуч. изображениями: Укажите путь к набору PNG-снимков.

    Файл аннотаций: Выберите текстовый файл(idl) с координатами «истинных» пешеходов.

    Сохранить модель в: Путь, где будет создан итоговый ZIP-архив модели.

    Параметры SVM:

        C (lambda): Параметр регуляризации. Чем меньше значение, тем шире разделяющая полоса.

        Итерации: Максимальное число шагов обучения.

        Bootstrapping: Если включено, программа проведет повторное обучение на «трудных» примерах (фон, похожий на человека).

    Действие: Нажмите кнопку «Study». Ход процесса отобразится в логе в нижней части окна.


**Детектирование:**
Применяется для обработки больших папок с изображениями и получения файла результатов.

    Выбор данных: Укажите папку с новыми изображениями и выберите ранее обученную модель (.zip).

    Результаты: Укажите путь к файлу результатов, куда будут записаны найденные координаты.

    Оценка качества (Ground Truth): Если у вас есть проверочный файл (GT), укажите его в соответствующем поле. Программа автоматически вычислит метрики Precision и Recall.

    Настройки окна:

        Шаг окна: Смещение окна поиска в пикселях (меньше шаг — выше точность, но медленнее работа).

        Порог: Минимальная уверенность классификатора для фиксации объекта.

        Подавление (NMS): Склеивает несколько рамок вокруг одного и того же человека в одну.

    Действие: Нажмите кнопку «Detect».

**Просмотр:**
Позволяет быстро протестировать модель на одном конкретном снимке.

    Нажмите «Загрузить изображение» и выберите файл.

    Нажмите «Загрузить модель» и выберите ваш ZIP-файл.

    Установите Порог (например, 0.00 или 0.50).

    Нажмите «Запустить детектор».

    Результат будет отрисован прямо на изображении зелеными рамками.
